File Menu
=========

New Project - Ctrl/Cmd + N
Open Project - Ctrl/Cmd + O
Close Project - Ctrl/Cmd + W
Save Project - Ctrl/Cmd + S
Save Project As... - <none on linux> should be Ctrl/Cmd + shift + S
Exit Natron - <none> suggest Ctrl/Cmd + Q

Edit Menu
=========

Preferences - <none> suggest Ctrl/Cmd + P
Undo - Ctrl/Cmd + Z
Redo - Ctrl/Cmd + shift + Z

Display Menu
============

New Viewer - Ctrl/Cmd + I
Toggle Full Screen - Meta+Ctrl/^Cmd + F

Options Menu
============

Project Settings - S

Render Menu
===========

Render all writers - F5
Render selected node - F7

Cache Menu
==========

Clear All Memory and Disk Caches - Ctrl/Cmd + shift + K


NodeGraph

Select all nodes - Ctrl/Cmd + A

Frame nodes - F
Display parameters links - shift+e
Expand and focus area - spacebar
Copy - Ctrl/Cmd+C
Cut - Ctrl/Cmd+X
Paste - Ctrl/Cmd+V
Duplicate - Alt+C
Clone - Alt+K
Declone - Alt+shift+K
Switch inputs - shift+X
Add Read node - R
Add Write node - W
Add Roto node - O
Add ColorCorrect node - C
Add Grade node - G
Add Transform node - T
Add Merge node - M
Add node by name - tab
